# Notex

**Anotações rápidas e organizadas para Foundry VTT.** Um substituto moderno para o Easy Notes.

Notex dá a cada usuário (jogadores e mestre) um espaço pessoal de anotações dentro do Foundry: um rascunho rápido sempre à mão e quantas notas você quiser, com editor rico, organizadas em pastas.

## Recursos

- **Rascunho rápido** — um campo de texto que salva sozinho, para anotar qualquer coisa na hora.
- **Múltiplas notas** com editor rico (ProseMirror nativo do Foundry), em modo visualização ou edição.
- **Pastas aninhadas** — organize as notas em pastas e subpastas, arrastando para reorganizar.
- **Fixar (pin)** notas importantes no topo.
- **Cor por nota** para identificação visual rápida.
- **Busca** por título e conteúdo.
- **Referências** — arraste uma nota para dentro de outra, ou copie o link; clicar navega dentro do próprio Notex.
- **Privacidade** — cada usuário vê apenas as próprias notas; o mestre gerencia tudo.
- **21 idiomas**, com escolha de idioma da interface por usuário (independente do idioma do mundo).
- Botão de acesso na barra de cena e na lista de jogadores.

## Instalação

No Foundry, vá em **Add-on Modules → Install Module** e cole o link do manifest:

```
https://github.com/Fernandogfg/notex/releases/latest/download/module.json
```

Ou procure por "Notex" no instalador de módulos do Foundry.

## Compatibilidade

- Foundry VTT v13 e v14 (verificado em 14.364).
- Não depende de sistema específico (funciona em qualquer sistema de jogo).

## Como usar

1. Ative o módulo no seu mundo.
2. Abra o Notex pelo botão na barra de controles de cena ou ao lado do seu nome na lista de jogadores.
3. As notas ficam organizadas em **Notex › seu nome** na aba de Diários.

## Idioma

Por padrão, o Notex segue o idioma do seu Foundry. Para mudar apenas para você, vá em **Configurações → Configurações do Módulo → Notex → Idioma do Notex**.
